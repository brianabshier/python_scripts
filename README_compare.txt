Text File Comparator

This Python script compares the contents of two text files line by line and identifies:
- Matching lines
- Lines unique to the first file
- Lines unique to the second file

Usage:
1. Run the script.
2. Enter the paths of the two text files when prompted.

Example:
python compare.py

Note: Ensure both files exist and are accessible.
