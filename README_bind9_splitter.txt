# DNS Zone Splitter

This Python script is designed to split a single input file containing multiple DNS zones into individual zone files based on the configuration pattern. Each DNS zone will be saved as a separate file in a specified directory.

## Prerequisites

- Python 3.x

## Usage

1. Clone the repository or download the script file.
2. Open a terminal or command prompt.
3. Navigate to the directory where the script is located.
4. Run the script using the following command:

python bind9_splitter.py input_file.txt
